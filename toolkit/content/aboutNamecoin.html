Namecoin (.bit) DNS

Namecoin provides decentralized DNS for .bit domains using the Namecoin blockchain.

Configuration

To enable .bit resolution, go to about:config and set:

  • network.namecoin.enabled → true
  • network.namecoin.electrumx_servers → ElectrumX server URL(s)

Test

Try navigating to http://testls.bit to test resolution.
